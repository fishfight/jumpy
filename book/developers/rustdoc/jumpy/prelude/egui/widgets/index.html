Module jumpy::prelude::egui::widgets

Expand description

Widgets are pieces of GUI such as Label, Button, Slider etc.

Example widget uses:

  • ui.add(Label::new("Text").text_color(color::red));
  • if ui.add(Button::new("Click me")).clicked() { … }

Expand description

A system parameter for editing the map.

MapManager provides an interface for editing the map contents easily while the match is running. This can be used both for manual map editing, and by algorithms for generating or randomizing maps.

Map generators implement the MapConstructor trait, which is given a MapManager to make its changes with.
